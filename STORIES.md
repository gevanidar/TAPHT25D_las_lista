
- Katalog

Som en användare,
vill jag kunna klicka på en knapp katalog,
så jag kan se all böcker som finns.

Som en Användare,
vill jag kunna klicka på en knapp mina böcker,
så jag kan kan se de böcker som jag har har favoritmarkerat.

Som en användare,
vill jag kunna favoritmarkera en bok,
så jag kan skapa en favorit lista över mina absoluta favoriter av alla böcker.

Som en användare,
vill jag kunna avmarkera en favoritbok,
så jag kan kan ånrga en felaktiv favoritmarkering.

Som en användare,
vill jag få en hint om vad som händer när jag tycker på en rad,
så jag kan få en visuell feedback av hur jag favoritmarkerar en bok utan att favoritmarkera boken.

- Lägg till bok

Som en användare,
vill jag kunna klicka på en knapp Lägg till bok,
så jag kan lägga till en bok som inte finns i katalogen över alla bäcker.

Som en användare,

- Statistik

Som en användare,
vill jag kunna klicka på en knapp Statistik
så jag kan se statistik över löslistan.

Som en användare,
vill jag kunna se statistik över antalet böcker,
så jag kan vet hur många böcker som finns i katalogen. 

Som en användare,
vill jag kunna se statistik över antalet favoritmarkerade böcker.
så jag kan vet hur många böcker som jag har favoritmarkerat.
